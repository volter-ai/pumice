# Welcome to uicommons

A server-optional, storage-agnostic display layer for an Obsidian-style vault.

The core idea is a [[Virtual FS]] with swappable [[Adapters]]. The renderer, the
[[Backlinks]] panel, and the [[3D Graph]] never know which backend they talk to.

Open this in two ways — see [[Two Modes]].
